Uebersicht

Das Auth Plugin kapselt Anmeldung, Rollenpruefung und Token-Verwaltung in klaren Hooks. Es bleibt nah an deinem Projektcode und vermeidet versteckte Magie.

Rollen

Saubere Berechtigungen fuer Redaktion, Admins und externe Integrationen.

Sessions

Konfigurierbare Token-Lebensdauer und sichere Defaults.

Features

  • JWT und OAuth2 Provider
  • Role-Based Access Control
  • Login Hooks fuer eigene Workflows
  • Audit Logs fuer sicherheitsrelevante Aktionen

Installation

cms plugin add auth
auth:
  provider: oauth2
  roles:
    - editor
    - admin
