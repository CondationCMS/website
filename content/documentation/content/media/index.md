# Media

Media formats are configured in [theme](/documentation/tempaltes-and-themes/theming) or [theme](/documentation/basics/site-config) config.

## Example configuration

```yaml
media:
  formats:
    - name: small
      width: 256
      height: 256
      format: webp
      compression: true
```
