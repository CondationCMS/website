# Extensions & Modules

This part is about extending the cms core.
In CondationCMS there are two ways to hook into the system pro create new or extend existing funktionality, extensions and modules.
Extensions are created via JavaScript inside a theme or a site. 
So they are relevant for project and theme developers.

Modules on the other side are created in Java and can user everythng that is accessible from Java.
Modules have to be installed into the server and activated for each theme or site who wants to use the module.
