# Getting started

To get the serer up and running and testing the contained demo project will just tke a few minutes

## 1. Download and unpack

Download the current release build for your platform from [github](https://github.com/CondationCMS/cms-server/releases).
Unpack the server into a directory of your choice.

## 2. Start the server

```shell
cd condation-server/
server.sh server start
```

Well done, you're ready. 
The condation-server is up and running.
If for some reasons it's not working, you may have some logs in the direct in the console or in some of the log files in the *logs* folder.

Read more about the [basics](/documentation/basics).

## 3. Access the frontend

Open your browser and type http://localhost:2020/

## 4. Access the manager

Open your browser and type http://localhost:2020/manager

Read more about the [manager](/documentation/manager).
